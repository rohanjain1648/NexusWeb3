# NexusWeb3

Next-generation Web3 dashboard with DeFi, NFTs, governance, and decentralized messaging — all in one beautiful interface.

## What technologies are used for this project?

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting Started

1. Clone the repository
2. Install dependencies with `npm i` or `bun install`
3. Run `npm run dev` or `bun dev` to start the development server
